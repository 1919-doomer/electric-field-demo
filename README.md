# 二维点电荷电场教学演示

一个运行在浏览器中的物理教学演示小程序，用于展示二维平面中多个点电荷产生的电场、电势，以及测试电荷在电场中的受力和运动。

项目使用 **React + TypeScript + Vite** 构建，渲染层基于 **HTML Canvas**，不依赖后端，适合课堂演示、物理可视化实验和教学辅助。

## 功能特性

- 添加正源电荷、负源电荷、正测试电荷、负测试电荷
- 拖拽电荷改变位置
- 编辑电荷量、质量、固定/可运动、是否作为源、是否作为测试电荷
- 显示电场强度热力图，颜色只表示 `|E|` 大小
- 显示电场方向箭头和可选电场线
- 对测试电荷显示电场力矢量 `F = qE`
- 对可运动电荷显示速度矢量和运动轨迹
- 鼠标悬停任意位置时显示坐标、`E`、`|E|`、`V`、探针电荷受力
- 支持开始、暂停、单步、重置、清空电荷、清除轨迹
- 支持调节时间步长 `dt`、最小作用距离 `r_min`、色标范围
- 支持边界反弹、穿出停止、环绕三种边界处理
- 内置可折叠的物理说明面板

## 主要物理模型

点电荷电场：

```text
E(r) = Σ k q_i (r - r_i) / |r - r_i|^3
```

电势：

```text
V(r) = Σ k q_i / |r - r_i|
```

测试电荷受力：

```text
F = q_test E
```

加速度：

```text
a = F / m
```

可运动电荷使用半隐式欧拉法积分：

```text
v += a * dt
x += v * dt
```

## 单位与数值近似

- 默认屏幕比例：`1 px = 0.002 m`
- 默认真实库仑常数：`k = 8.9875517923e9 N·m²/C²`
- 程序使用 `simulationK = 0.02 * k` 做课堂可视化缩放，公式结构保持不变
- 电荷量 UI 使用 `μC`
- 质量 UI 使用 `g`
- `r_min` 用于限制最小作用距离，避免点电荷模型在 `r -> 0` 时发散
- 热力图使用网格采样和 `log10(1 + |E|)` 色标，避免极大值完全支配颜色
- 速度上限和加速度上限是数值稳定保护，不是基本物理定律
